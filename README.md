# Main Pages
- ScoreBoard(Main/Default)
- Add user/delete user: no login/signup. More like a add user, and add passkey if wanted.
- Media Room: Upload clips to determine improvements need like form, tossing speed, etc. Option to save media to database
- Past Games: Shows game history for the past of a max of 3 weeks, option to permanently save game
